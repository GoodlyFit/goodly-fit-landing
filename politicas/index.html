Politicas de privacidad

Documento provisorio. El texto final sera reemplazado cuando este disponible.

1. Alcance

Esta pagina describe de forma provisoria como Goodly Fit puede recopilar, usar y proteger informacion vinculada al uso de sus sitios, formularios, agenda y comunicaciones comerciales.

2. Informacion recopilada

Podemos recopilar datos enviados por formularios, datos de contacto, informacion del negocio, preferencias comerciales, datos tecnicos basicos y eventos de navegacion necesarios para operar, medir y mejorar la experiencia.

3. Uso de la informacion

La informacion puede utilizarse para responder consultas, coordinar reuniones, brindar servicios, realizar seguimientos, mejorar campañas, medir rendimiento y mantener registros operativos dentro de herramientas comerciales autorizadas.

4. Herramientas externas

Goodly Fit puede utilizar plataformas de analitica, publicidad, CRM, calendario, automatizacion y mensajeria para operar el servicio y gestionar oportunidades comerciales.

5. Derechos y contacto

El usuario puede solicitar la revision, actualizacion o eliminacion de sus datos escribiendo a los canales oficiales de Goodly Fit.
